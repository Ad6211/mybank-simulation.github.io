## 📊 À propos

Ce projet permet de simuler différents comportements bancaires et d'observer leur impact.

- 💳 Gestion d'argent simulée  
- 📈 Visualisation simple  
- 🌐 Accessible directement dans le navigateur

2. Lance la simulation dans ton navigateur
3. Teste différents scénarios
